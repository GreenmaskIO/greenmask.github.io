# Port Greenmask.io Website to Docusaurus

We have the folowing:

* [Greenmask website](https://www.greenmask.io/) that is publised to the internet
* [A section of figma project](https://www.figma.com/design/hBAgs4wT3FD0IeW6yr2MpC/greenmask?node-id=770-1561&t=QaqRQylCjLI7iluq-4)
  that basically describes a common blocks and styles of the website
* This repository with initialized docusaurus project

We need to port the website to docusaurus, so we can maintain it in a more structured way and have a better experience
for contributors. We need to save original style of the website, but we can make some improvements if needed. 
You need to use the figma project as a source of truth for styles and blocks, but you can also refer to the original 
website for some details. Figma does not have some images. 

Let's focus for now only on the main page of the website. You could implement similar header and footerlike on the
original website, but skip the other pages.
